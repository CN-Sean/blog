# 组合模式 Composite

### 特点：在不改变接口的情况下，装饰器通过组合方式引用对象，并由此在保持对象功能的基础上给对象加上新功能。

### 用处：当需要不影响现有类来增加新的功能时，可以考虑装饰模式，它可以动态透明的给对象增加功能。
